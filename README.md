# dressgame

O Dress Game é um joguinho de navegador que eu fiz usando **HTML, CSS e JavaScript** para montar looks do jeito que quiser.

Há um panda na tela e as peças, como chapéu, camiseta e calça, podem ser trocadas ao clicar. As imagens mudam na hora, permitindo testar várias combinações.

Criei esse projeto para praticar e desenvolver algo simples e criativo.

A estética foi inspirada em jogos antigos, como os do **Game Boy Advance**, trazendo uma sensação dos anos 2000.

##  Inspiração

Esse projeto foi inspirado no trabalho de Nashallery.

Usei a ideia como referência, mas toda a implementação foi feita por mim do zero, sem copiar código.

As artes também foram feitas por mim, com exceção do plano de fundo.
